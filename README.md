# World Map Timeline Tool for Minecraft (WMTT4MC)
World Map Timelapse Tool for Minecraft - generates a timeline of your Minecraft worlds based on sequential world backup files. Can also generate high-quality, high-resolution maps of any world backup

## Downloads (Official Binaries)

**Official Windows and macOS binaries are distributed ONLY via this repository’s GitHub Releases page.**  
If you downloaded WMTT4MC from anywhere else, it is **not** an official build.

- Go to the **Releases** page and download the latest version for your OS.

### Security prompts (unsigned binaries)
WMTT4MC is currently distributed **without** code signing/notarization. Your OS may show warnings:

- **macOS**: You may need to right-click the app and choose **Open**, then confirm.
- **Windows**: You may see a SmartScreen warning; choose **More info** → **Run anyway** (if you trust the source).

To verify integrity, you can compare SHA256 checksums (if provided in the Release).
